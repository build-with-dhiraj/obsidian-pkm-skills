## Overview

The PARA method organizes information into four categories: Projects, Areas,
Resources, and Archives. It is a top-down organizational system popularized by
Tiago Forte. In this demo vault PARA is deliberately over-connected: many
source notes point at it, making it a dominant hub the auditor should flag.
