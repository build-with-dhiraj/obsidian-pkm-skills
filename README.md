# obsidian-brain-eval

> Read-only command-line tool that measures whether your Obsidian vault retrieval actually works. Generates a gold-set from your notes, scores any retrieval backend against Recall@10, and tells you PASS or BELOW TARGET. Pluggable backends (BM25 zero-infra, LanceDB hybrid). Works in Claude Code, Cursor, Gemini CLI, and Codex.
